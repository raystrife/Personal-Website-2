Selenium Test

feat. Playwright framework

Everyone who deals with automation test knows what Selenium is. For me, Selenium is one of the first automation tool I learned to test web pages. It is an open-source tool, which additionally is easy to understand and use! To demonstrate my proficiency about Selenium, I wrote a Selenium test script in order to test the previous version of my personal website. Please take a look at my script in my repository!

However, I discovered a better automation tool that suits my need the best: Playwright! When I was using Selenium, one of my pet peeve is that the Selenium's methods will immediately fails the test if they are unable to find the elements on the webpage. Playwright's methods, however, come with the functionality to wait for a period of time for the elements to show up! Additionally, Playwright comes with built-in methods that deal with executing various actions on the usual web elements, such as selecing from a dropdown or pressing a radio button. Playwright will be my favorite automation tool for a while!
